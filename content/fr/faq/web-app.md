# FAQ de la web app

:::details Ai-je besoin de l'app mobile pour utiliser la web app ?

Oui. Votre compte Peach vit dans l'app mobile, la web app s'y connecte. Téléchargez l'app, créez votre compte en quelques secondes (et sauvegardez-le !), puis connectez votre navigateur en scannant le code QR.
:::

:::details Peach détient-il mes bitcoins quand j'échange depuis mon ordinateur ?

Non. Peach est auto-conservateur partout : chaque échange est protégé par un séquestre multisig et vos bitcoins vont directement dans votre propre portefeuille. Le grand écran ne change rien à qui détient les clés.
:::

:::details Que puis-je faire depuis mon ordinateur ?

Parcourir le carnet d'ordres complet, envoyer et accepter des demandes d'échange, créer des offres d'achat et de vente et suivre vos échanges en cours : tout ce que vous faites sur mobile, à l'exception de l'opération de signature. Consultez [l'article de blog sur le sujet](/fr/blog/peach-web-launch/) pour plus de détails !
:::

:::details Se connecter avec un code QR est-il sûr ?

Oui, cela fonctionne comme WhatsApp Web. Le scan relie votre navigateur au compte de votre téléphone. Rien n'est partagé avec des tiers, et la session prend fin dès que vous fermez la fenêtre, vous pouvez aussi vous déconnecter à tout moment. Vérifiez bien que vous êtes sur la bonne URL (web.peachbitcoin.com) !
:::

:::details La web app coûte-t-elle quelque chose en plus ?

Non. La web app est gratuite et les frais de transaction sont exactement les mêmes que dans l'app mobile.
:::

:::details Comment déconnecter une session web ?

Fermer la fenêtre du navigateur vous déconnecte automatiquement, rien ne reste connecté, ce qui rend la web app sûre même sur un ordinateur partagé ou public. Vous pouvez aussi vous déconnecter à tout moment avec le bouton en haut à droite.
:::

:::details Quels navigateurs puis-je utiliser ?

La web app fonctionne dans tout navigateur moderne et à jour, par exemple Firefox, Chrome, Brave, Safari ou Edge, sous Linux, macOS et Windows. Aucune extension ni installation n'est nécessaire : ouvrez simplement web.peachbitcoin.com. Pour votre vie privée, préférez un navigateur de confiance et gardez-le à jour.
:::

:::details Que se passe-t-il si mon téléphone est hors ligne ?

Votre téléphone détient les clés de votre compte : les actions qui nécessitent une signature, comme la libération du séquestre, requièrent donc votre téléphone. Vous pouvez continuer à parcourir le carnet d'ordres dans la web app, mais pour finaliser les actions sensibles d'un échange, remettez votre téléphone en ligne et confirmez depuis celui-ci. Vos fonds dans le séquestre multisig ne sont jamais en danger pendant que votre téléphone est hors ligne.
:::

:::details Puis-je connecter plusieurs navigateurs ?

Oui, vous pouvez connecter autant de navigateurs que vous le souhaitez : scannez simplement le code QR sur chaque ordinateur. Chaque session ne dure que jusqu'à la fermeture de la fenêtre ou votre déconnexion, rien ne reste connecté derrière vous.
:::

:::details Pourquoi dois-je confirmer certaines actions sur mon téléphone plutôt que dans le navigateur ?

C'est l'auto-conservation par conception. Vos clés privées ne quittent jamais votre téléphone : tout ce qui déplace des bitcoins, comme la signature de la libération du séquestre, doit donc être confirmé dans l'app mobile. Le navigateur est une fenêtre pratique sur votre compte ; votre téléphone reste le coffre-fort.
:::

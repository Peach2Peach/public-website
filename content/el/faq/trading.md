# Συχνές Ερωτήσεις Συναλλαγών

:::details Πώς μπορώ να είμαι σίγουρος ότι θα πάρω τα bitcoin / τα χρήματα;

Όταν δημιουργείτε μια προσφορά πώλησης ή όταν ταιριάζετε με μια προσφορά αγοράς, ο πωλητής στέλνει τα bitcoin σε μια διεύθυνση που ελέγχεται από αυτόν και την Peach: τα bitcoin μπορούν να μετακινηθούν μόνο εάν υπογράψουν και ο πωλητής και η Peach. Αυτό διασφαλίζει ότι:

- Ο πωλητής δεν μπορεί να μετακινήσει τα bitcoin (πίσω) μόνος του
- Η Peach δεν μπορεί να κλέψει τα bitcoin
- Ο αγοραστής δεν παίρνει τα bitcoin μέχρι να γίνει η πληρωμή
- Ο πωλητής μπορεί να πάρει πίσω τα bitcoin αν ο αγοραστής δεν απαντήσει

Αν η συναλλαγή δεν λυθεί κανονικά, η διεύθυνση αυτή περνά αυτόματα στον πλήρη έλεγχο της Peach μετά από περίπου 30 ημέρες (πιο συγκεκριμένα: όταν εξορυχθούν 4320 bitcoin blocks). Αυτό διασφαλίζει ότι:

- Ο αγοραστής μπορεί να πάρει τα bitcoin αν αποδείξει ότι έκανε την πληρωμή, αλλά ο πωλητής δεν απαντά
- Τα bitcoin δεν θα «κολλήσουν» αν συμβεί κάτι στον πωλητή

Αυτό είναι το πιο σημαντικό μέρος για την ασφάλεια της συναλλαγής σας. Επιπλέον, υπάρχει και το περίπλοκο σύστημα φήμης μας που σας βοηθά να εντοπίσετε άτομα που χρησιμοποιούν αξιόπιστα την Peach εδώ και καιρό.
:::

:::details Γιατί υπάρχει όριο συναλλαγών;

Οι ελβετικοί κανονισμοί αναφέρουν ότι ένα άτομο μπορεί να αγοράσει έως 1000 CHF bitcoin την ημέρα χωρίς να παρέχει τα στοιχεία ταυτότητας στον πωλητή. Επειδή δεν θέλουμε να βρεθούμε στη φυλακή, εφαρμόζουμε αυτό το όριο στην εφαρμογή.

Όλες οι λεπτομέρειες πληρωμής σας αποθηκεύονται στο τηλέφωνό σας, επομένως δεν μπορούμε να τις δούμε. Αυτό που μπορούμε να δούμε είναι ένα hash\* του ID του τηλεφώνου σας και των στοιχείων πληρωμής σας. Αυτό μας επιτρέπει να μπλοκάρουμε συναλλαγές που υπερβαίνουν το προσωπικό σας όριο.

\* Ένα hash είναι δεδομένα που έχουν καταστεί μη αναγνωρίσιμα, παρόμοια με την κρυπτογράφηση. Τα ίδια δεδομένα οδηγούν πάντα στο ίδιο hash. Αυτό σημαίνει ότι δεν γνωρίζουμε ποια είναι τα δεδομένα, αλλά μπορούμε να δούμε αν χρησιμοποιούνται δύο φορές.
:::

:::details Υπάρχει τρόπος να αγοράσω/πουλήσω περισσότερα από το όριο συναλλαγών;

Αν είστε πωλητής με υψηλό όγκο, στείλτε μας ένα email στο [$contactEmail$](mailto:$contactEmail$)!
Θα σας ζητηθεί να ακολουθήσετε μια διαδικασία KYC και τα όρια σας θα καταργηθούν.
:::

:::details Ποιες είναι οι χρεώσεις συναλλαγών στην Peach;

Η Peach χρεώνει τον αγοραστή με προμήθεια 2% του όγκου συναλλαγών. Όταν κάνετε συναλλαγές στην Peach, εκτελείτε συναλλαγές στο δίκτυο Bitcoin, πράγμα που συνεπάγεται χρεώσεις συναλλαγών. Στο τέλος κάθε συναλλαγής μπορείτε να δείτε αναλυτικά όλες τις χρεώσεις, που μπορεί να μοιάζουν κάπως έτσι:

![Ανάλυση Συναλλαγής](/img/faq/trading/TradeBreakdowns.png)
:::

:::details Πώς μπορώ να ακυρώσω μια προσφορά ή μια συναλλαγή;

Μπορείτε να ακυρώσετε τις προσφορές και τις συναλλαγές σας κάνοντας κλικ στο κόκκινο Χ στην κορυφή της οθόνης, όποτε είναι διαθέσιμο:

![Ακύρωση Συναλλαγής](/img/faq/trading/cancel.png)

Ωστόσο, αυτό συχνά έχει συνέπειες. Πριν ταιριάξετε με κάποιον, μπορείτε να ακυρώσετε ανά πάσα στιγμή. Αφού ταιριάξετε, η φήμη σας θα επηρεαστεί αρνητικά. Επιπλέον, ως πωλητής θα χρειαστεί να ζητήσετε από τον αγοραστή άδεια για να ακυρώσετε τη συναλλαγή. Μπορεί να έχει ήδη κάνει την πληρωμή!
:::

:::details Γιατί έλαβα λιγότερα sats από όσα περίμενα;

Η Peach χρεώνει 2% προμήθεια στον αγοραστή, που σημαίνει ότι θα λάβετε λιγότερα sats από την ονομαστική αξία της συναλλαγής. Επιπλέον, θα πρέπει να πληρώσετε χρεώσεις δικτύου Bitcoin. Για παράδειγμα, η συναλλαγή σας μπορεί να μοιάζει έτσι:

![Ανάλυση Αγοράς](/img/faq/trading/TradeBreakdownBuy.png)
:::

:::details Τι γίνεται αν δεν θέλω να χρησιμοποιήσω το πορτοφόλι Peach για πληρωμή / επιστροφή;

Φυσικά, μπορείτε να χρησιμοποιήσετε το δικό σας πορτοφόλι αν θέλετε. Ωστόσο, συνιστούμε ανεπιφύλακτα τη χρήση του πορτοφολιού Peach, καθώς είναι ο πιο εύκολος τρόπος για να κάνετε μια συναλλαγή. Στη συνέχεια, μπορείτε να στείλετε τα χρήματα σε οποιοδήποτε άλλο πορτοφόλι.

Αν θέλετε να προσθέσετε το δικό σας πορτοφόλι, μπορείτε να απενεργοποιήσετε το «πληρωμή στο πορτοφόλι Peach» και να ορίσετε μια προσαρμοσμένη διεύθυνση πληρωμής:

![Απενεργοποίηση Πορτοφολιού](/img/faq/trading/disablewallet.png)

Κατά τη διάρκεια της συναλλαγής, θα χρειαστεί να υπογράψετε ένα μήνυμα που να αποδεικνύει ότι ελέγχετε αυτό το πορτοφόλι, σύμφωνα με τους ελβετικούς κανονισμούς.

Θα προσθέσουμε σύντομα υποστήριξη για xpub, αλλά προς το παρόν πρέπει να αλλάξετε χειροκίνητα τη διεύθυνση αν δεν θέλετε να την επαναχρησιμοποιήσετε.
:::

:::details Πώς υπολογίζεται η τιμή του Bitcoin στην Peach;

Η τιμή BTC που δείχνουμε στην Peach είναι ένας μέσος όρος των τιμών BTC σε κεντρικά ανταλλακτήρια.
:::

:::details Τι συμβαίνει με τις τιμές νομισμάτων σε χώρες με υψηλό πληθωρισμό όπως η Αργεντινή, η Βενεζουέλα κ.λπ.;

Τα νομίσματα με υψηλό πληθωρισμό έχουν μεγάλη μεταβλητότητα, επομένως οι τιμές στα διάφορα ανταλλακτήρια μπορεί να διαφέρουν. Η Peach δείχνει την τιμή σύμφωνα με έναν μέσο όρο BTC τιμών από διαφορετικές πηγές.
:::

:::details Πώς μπορώ να επισπεύσω μια συναλλαγή που έχει κολλήσει λόγω χαμηλών χρεώσεων εξόρυξης;

Εξαρτάται από τον τύπο της συναλλαγής. Εδώ είναι μια λίστα με όλες τις συναλλαγές που μπορούν να συμβούν στην Peach και τις λύσεις για να αυξήσετε τα τέλη:

1. Συναλλαγή για τη χρηματοδότηση του escrow για δημοσίευση προσφοράς πώλησης ή για ταιριάξιμο με προσφορά αγοράς

- Αν χρηματοδοτήσατε το escrow από το πορτοφόλι Peach, μπορείτε να χρησιμοποιήσετε RBF (Replace-By-Fee) και να αυξήσετε τα τέλη
- Αν χρηματοδοτήσατε το escrow από εξωτερικό πορτοφόλι, πρέπει να ελέγξετε αν το πορτοφόλι υποστηρίζει RBF

2. Συναλλαγή απελευθέρωσης από το escrow (αγορά Bitcoin)

- Αν η διεύθυνση λήψης είναι από το πορτοφόλι Peach, μπορείτε να κάνετε ανάληψη με υψηλότερα τέλη (Ρυθμίσεις > Χρεώσεις Δικτύου) – τεχνική CPFP
- Αν η διεύθυνση λήψης είναι από εξωτερικό πορτοφόλι, μπορείτε επίσης να χρησιμοποιήσετε την τεχνική CPFP αν την υποστηρίζει το πορτοφόλι σας

3. Συναλλαγή αποστολής από το πορτοφόλι Peach σε άλλο πορτοφόλι

- RBF (Replace-By-Fee) από το πορτοφόλι Peach στις λεπτομέρειες συναλλαγής!
  :::

:::details Τι είναι το GroupHug;
Το GroupHug είναι απλώς ο όρος που δώσαμε στην ομαδοποίηση συναλλαγών από διαφορετικούς χρήστες ώστε να αποφύγουν ατομικά τέλη. Για μια πιο λεπτομερή εξήγηση, δείτε το [blog post](https://peachbitcoin.com/blog/group-hug).
:::

:::details Αν έχω μία μόνο προσφορά αγοράς σε εξέλιξη θα εκταμιευτεί αμέσως;

Όχι, η πληρωμή σας θα προστεθεί σε μια ουρά, περιμένοντας εκταμίευση. Η εκταμίευση γίνεται όταν αρκετοί χρήστες συμμετάσχουν στο batch. Ο αριθμός των απαιτούμενων συμμετεχόντων φαίνεται στις πληροφορίες εκκρεμών πληρωμών. Εκεί μπορείτε να δείτε πόσες θέσεις του τρέχοντος batch έχουν καλυφθεί και ένα ETA που σας δείχνει τον μέγιστο χρόνο αναμονής.
:::

:::details Πώς λειτουργεί αν έχω πολλές προσφορές αγοράς ταυτόχρονα;

Όπως αναφέρθηκε, οι πληρωμές σας θα προστεθούν στην ουρά για να ομαδοποιηθούν με άλλους συμμετέχοντες.
:::

:::details Υπάρχει όριο στον αριθμό συμμετεχόντων στο batch;

Όχι, τα batches μπορούν να περιέχουν και περισσότερους συμμετέχοντες από το ελάχιστο. Δεν είναι cutoff αλλά όριο. Μόλις επιτευχθεί το ελάχιστο, παίρνουμε όλα τα psbts και τα ομαδοποιούμε ώστε να γίνει η συναλλαγή και να μειωθούν τα τέλη για όλους.
:::

:::details Πώς να υπογράψω μια εξωτερική διεύθυνση;
Ακολουθήστε αυτά τα βήματα για να υπογράψετε τη διεύθυνση λήψης όταν αγοράζετε Bitcoin σε εξωτερικό πορτοφόλι:

_Σημείωση: Τα πρώτα 2 βήματα είναι χρήσιμα αν θέλετε **πάντα** να λαμβάνετε τα χρήματά σας σε εξωτερικές διευθύνσεις. Αν το θέλετε μόνο μία φορά ή μερικές φορές, ξεκινήστε από το βήμα 3._

1. Μεταβείτε στις ρυθμίσεις

- Απενεργοποιήστε το πορτοφόλι Peach
- Μεταβείτε στη διεύθυνση πληρωμής

2. Επικολλήστε τη νέα διεύθυνση λήψης

3. Προχωρήστε στη διαδικασία για να δημοσιεύσετε την προσφορά αγοράς και βεβαιωθείτε ότι επιλέξατε εξωτερική διεύθυνση πριν τη δημοσίευση (πατήστε στο μικρό εικονίδιο πορτοφολιού επάνω δεξιά στην οθόνη σύνοψης).

4. Μόλις επιβεβαιώσετε την προσφορά αγοράς, θα εμφανιστεί το μήνυμα για υπογραφή. Αντιγράψτε το και επιστρέψτε στο πορτοφόλι σας.

5. Αναζητήστε την επιλογή "sign/verify"\* και επικολλήστε:

- τη διεύθυνση λήψης
- το μήνυμα της Peach

6. Πατήστε "υπογραφή" και θα εμφανιστεί η υπογραφή. Αντιγράψτε την.

7. Επικολλήστε την υπογραφή στο πορτοφόλι Peach και επιβεβαιώστε.

8. Η προσφορά σας δημοσιεύτηκε.

_\* Προειδοποίηση: δεν υποστηρίζουν όλα τα πορτοφόλια την επιλογή υπογραφής._
Η Peach προτείνει Blue Wallet, Sparrow ή Samourai Wallet. Άλλες επιλογές είναι Ledger και Trezor (Hardware Wallets), Bitcoin Core και Electrum.

Θα βρείτε επίσης ένα βίντεο με οδηγίες για την υπογραφή μηνύματος χρησιμοποιώντας Blue Wallet στο Youtube: [https://youtu.be/d3STuVfFWfQ](https://youtu.be/d3STuVfFWfQ)
:::

:::details Πώς να χρησιμοποιήσω CPFP για να επιταχύνω κολλημένες συναλλαγές;

Ακολουθήστε τα βήματα σε αυτό το βίντεο: [https://www.youtube.com/watch?v=24OtQkL0CxU](https://www.youtube.com/watch?v=24OtQkL0CxU) για να επιταχύνετε τις συναλλαγές που έχουν κολλήσει χρησιμοποιώντας CPFP μέσα στην εφαρμογή Peach.
:::

:::details Πώς λειτουργεί η χρηματοδότηση πολλών προσφορών πώλησης από εξωτερικό πορτοφόλι;

Όταν θέλετε να χρηματοδοτήσετε πολλές προσφορές πώλησης ταυτόχρονα, η εφαρμογή δημιουργεί μια ενδιάμεση διεύθυνση από το πορτοφόλι Peach. Μόλις τα bitcoin φτάσουν σε αυτήν τη διεύθυνση, δημιουργείται νέα συναλλαγή για κάθε escrow. Για παράδειγμα, αν θέλετε να χρηματοδοτήσετε 5 προσφορές πώλησης, θα σταλούν 5 συναλλαγές σε διαφορετικές διευθύνσεις escrow.
:::
